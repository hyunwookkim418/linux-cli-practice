# Linux CLI Practice

Hands-on Linux CLI training focused on **real data center troubleshooting**.

## Structure
Each day includes:
- CLI Commands (8–12)
- Key Concepts
- Real DC Scenario
- Verification
- Troubleshooting Steps

## Progress
- Day 01 – File Operations  
- Day 02 – Server Health Check  
- Day 03 – Search  
- Day 04 – Process Monitoring  

## Goal
Build fast, practical skills to handle:
- Server slowness
- Disk issues
- Process failures
- Basic system troubleshooting

## Usage
Run commands → repeat → understand output
